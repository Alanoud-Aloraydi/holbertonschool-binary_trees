# Binary Trees

This project focuses on the implementation and manipulation of binary trees in C.

It covers the creation of tree nodes, insertion and deletion operations, and different traversal methods used to explore tree structures. In addition, the project explores key properties of binary trees such as height, depth, size, number of leaves and nodes, and balance factor.

It also introduces structural classifications of binary trees, including full and perfect trees, and relationships between nodes such as parent, sibling, and uncle.

The objective is to develop a solid understanding of tree-based data structures and their practical use in problem solving and algorithm design.
